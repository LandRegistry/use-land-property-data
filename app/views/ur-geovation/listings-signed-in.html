Restrictive Covenants

Cost:
Updated:
Monthly
Licence:
3 licences available
File size and type:
ZIP 1.17GB, CSV
API available:
Yes

Download data about all registered land and property that have restrictive covenants. A restrictive covenant is a contract between 2 landowners. One landowner promises the other landowner that they will not carry out certain acts on their land.

The data shows if there's a restrictive covenant but it does not give details of what it is.

To access this dataset, you'll need to chose a licence based on how you're going to use the data and create an account.
