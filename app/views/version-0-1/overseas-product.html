The dataset

Download data about non-UK companies that own land or property in England and Wales. The data includes (where available) company name, type of company, registration number, address and price paid. This dataset was originally called Overseas companies ownership data (OCOD).

Dataset licence

Your use of the National Polygon Service is governed by the terms and conditions of a licence.

We use data from Ordnance Survey to prepare polygons and the dataset. This data including the UPRN and polygon ID is under licence

You’ll need to contact Ordnance Survey and enter into a licence agreement with them to use this data.

We can only supply this dataset to you once you’ve entered into this agreement.

Access the data

Download the data

The dataset is updated on the second working day of the month. The data includes an ESRI shapefile and CSV files. You can download the

  • complete file - this is a complete file of the current data and contains all of the changes that happened during the month
  • change only file - this only includes data that has changed since the previous month’s release
  • validation file - this checks that the data has been downloaded correctly

The Title Number and UPRN Look Up dataset does not have a change only file.

Access the data with an API

You’ll get an API key when you’ve:

  • agreed to the licence
  • paid for the data
  • signed in to your account

For more information, read our guides on accessing data with the API and API technical details.
