The dataset

This data shows registered land and property in England and Wales owned by UK companies.

The data does not show land or property owned by:

  • private individuals​
  • overseas companies
  • charities

We cannot show records that are being updated. For example, a company buys a property and tells us about it on 20 June 2019. We finish updating the record on 12 July 2019. You will not be able to download this data until the dataset is next updated on the second working day of August.

Data quality

Where the information comes from

We get information from:

  • citizens​
  • corporate bodies
  • conveyancers​
  • deeds​
  • other documents

Inconsistencies in the data​

The data comes from different sources. This means that not everything is written the same way. For example, one address may say '24 The Willows' and another might be written as 'Field south of N14'.

You may also see:

  • spelling errors or typos​
  • descriptions instead of addresses (land being developed may not have an address)
  • partial postcodes or no postcode at all​
  • mistakes in company registration numbers​
  • no data for date of registration
  • no data for company registration number​
  • 2 different companies with the same registration number

Inconsistencies happen because of human error or because we do not have the data. For example, companies who bought property before 1996 did not need to tell us their registration number.

Access the dataset

You’ll need to create an account or  sign in to your account, and agree to the data licence.

Download the data

The dataset is updated on the second working day of the month and contains data for the previous month. For example, files published in April show data from March. The data includes CSV files. You can download the:

  • complete file – this is a complete file of the current data and contains all of the changes that happened during the month
  • change only file – this includes data that has changed since the previous month’s release
  • validation file - this checks that the data has been downloaded correctly

Access the data with an API

You'll get an API key when you've:

  • created your account
  • signed in
  • agreed to the data licence

For more information, read our guides on accessing data with the API  and API technical detail.

How the data can be used

This data has a licence which explains how you can use it. If you're not sure that you can use the data in the way you need, email address.management@royalmail.com to check.

The licence sets out the conditions for the re-use of data (in this case, addresses) which contain third party intellectual property rights. Property addresses contained within the Overseas companies that own property in England and Wales data are processed against Ordnance Survey's AddressBase Premium product, which incorporates Royal Mail's PAF® database (Address Data).
