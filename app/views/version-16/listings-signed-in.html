INSPIRE Index Polygon spatial data

Cost:
Free of charge
Updated:
Monthly
Licence:
Open Government Licence
File size and type:
Average 15MB GML (estimated)
API available:
No

View or download HM Land Registry INSPIRE Index Polygons. They show the indicative position of registered freehold properties in England and Wales. The data is shown as shapes on a map. You’ll need a Geographical Information System (GIS) to view downloaded polygons.

National Polygon Service

Cost:
£20,000 plus VAT a year
Updated:
Monthly
Licence:
Restricted
File size and type:
ZIP 7.5GB, CSV and shape file
API available:
Yes

Download data about registered land and property that shows:

  • indicative shape and position of boundaries
  • legal interests and class of title
  • unique property reference numbers

It costs £20,000 plus VAT a year and you’ll need to create an account to download this data.
