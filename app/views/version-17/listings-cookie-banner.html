HM Land Registry Use land and property data

Access datasets about all registered land and property in England and Wales.

This is bulk data suitable for business and professional use.

You could use it in:

  • research
  • your work
  • online data services or products

Most of the data is free to access but you may need to create an account and sign a licence agreement first.

Some datasets are large. You may need specialist software to open them.

! Warning Sign in to Kahootz if you normally use it to access Overseas or UK companies datasets.

We'll be moving Kahootz users to this service from February 2020.

INSPIRE Index Polygon spatial data

Cost:
Free of charge
Updated:
Monthly
Licence:
Limited
File size and type:
average 15MB, GML
API available:
No

View or download HM Land Registry INSPIRE Index Polygons which show the indicative position of registered freehold properties in England and Wales. The data is shown as shapes on a map. You’ll need a Geographical Information System (GIS) to view the polygons.

National Polygon Service

Cost:
£20,000 plus VAT a year
Updated:
Monthly
Licence:
Restricted
File size and type:
ZIP 7.5GB, CSV and shape file
API available:
Yes

Download data about registered land and property that shows:

  • indicative shape and position of boundaries
  • legal interests and class of title
  • unique property reference numbers

It costs £20,000 plus VAT a year and you’ll need to create an account to download this data.
