HM Land Registry Use land and property data

Access large datasets about land and property in England and Wales.

Most of this data is free to download but you may need to create an account and sign a licence agreement first.

1862 Act Register

Cost:
Free of charge
Updated:
Never
Licence:
Open Government Licence
File size and type:
1.5MB, JPG
API available:
No

Download historical data about people and properties recorded on the 1862 Act Register. This was the first land register that HM Land Registry created and was the government’s first attempt to record property ownership information. It shows the owner name and property they owned.

INSPIRE Index Polygon spatial data

Cost:
Free of charge
Updated:
Monthly
Licence:
Limited
File size and type:
average 15MB, GML
API available:
No

View or download HM Land Registry INSPIRE Index Polygons which show the indicative position of registered freehold properties in England and Wales. The data is shown as shapes on a map. You’ll need a Geographical Information System (GIS) to view the polygons.

Download data about non-UK companies that own land or property in England and Wales. This dataset was originally called Overseas companies ownership data (OCOD).

You’ll need to create an account to download this data.

National Polygon Service

Cost:
£20,000 plus VAT a year
Updated:
Monthly
Licence:
Restricted
File size and type:
ZIP 7.5GB, CSV and shape file
API available:
Yes

Download data about registered land and property that shows:

  • indicative shape and position of boundaries
  • legal interests and class of title
  • unique property reference numbers

It costs £20,000 plus VAT a year and you’ll need to create an account to download this data.
